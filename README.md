𝐎𝐯𝐞𝐫𝐯𝐢𝐞𝐰
LifeBridge Pakistan is a disaster-response web application designed to help coordinate relief efforts during major crises — cyclones, earthquakes, and floods. It brings together live mapping, an offline-capable AI assistant, shelter and route information, supply-demand matching, a safety registry, and volunteer task management into a single, unified interface.

The platform supports multiple real-world disaster scenarios, allowing responders to switch context instantly between:

- Karachi Cyclone (Category 4)
- Kashmir Earthquake (M7.6)
- Sindh River Monsoon Floods

𝗠𝘂𝗹𝘁𝗶-𝗦𝗰𝗲𝗻𝗮𝗿𝗶𝗼 𝗗𝗶𝘀𝗮𝘀𝘁𝗲𝗿 𝗦𝘄𝗶𝘁𝗰𝗵𝗲𝗿
Switching scenarios from the header dropdown dynamically:
- Recenters the Leaflet map to the affected region (Karachi, Muzaffarabad, or Sukkur)
- Redraws shelters, field clinics, and hazard markers at real Pakistani coordinates
- Updates threat badges, tickers, and dashboard metrics
- Reconfigures the Safe Route Finder's routing nodes
- Resets the AI Agent's greeting and suggestion chips for the new context

𝗗𝘆𝗻𝗮𝗺𝗶𝗰 𝗦𝗵𝗮𝗿𝗲𝗱 𝗘𝗹𝗲𝗺𝗲𝗻𝘁 𝗥𝗼𝘂𝘁𝗶𝗻𝗴
A custom DOM-repositioning system (in app.js) moves the "#map" and ".chat-container" elements between views using "appendChild()" and "map.invalidateSize()"- preserving map state and markers seamlessly as users navigate between Dashboard, Map, and AI Agent tabs.
